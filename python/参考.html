# range

range 是 Python 中的一个内置函数，用于生成一个整数序列。它主要用于循环中，特别是 for 循环，以控制循环的次数，如何使用？前往

range(start, stop, step)

参数

  • start 整数序列的起始位置
  • stop 整数序列的结束位置
  • step 整数序列的步长，默认为 1
