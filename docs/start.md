## 分类搜索

随着知识的不断积累，普通的搜索方式已经不足以满足需求，搜索时会出现许多文章名称重复的情况，这时就需要使用这个分类搜索，这种搜索方式可以定位在一个领域中去搜索文章，有效的缩小了搜索范围，减少了文章名称重复的情况

**搜索格式**：`z/类型/关键词`，例如搜索**c语言**中与**变量**相关的内容，输入`z/c/变量`

|种类|检索符|种类|检索符|
|---|---|---|---|
|HTML|z/html/|C| z/c/|C++|z/cpp/|
|CSS|z/css/|Python|z/py/|
|JavaScript|z/js/|数学|z/math/|
|Vue.js|z/vue/|物理|z/phys/|
|TypeScript|z/ts/|||
