# 引用文献格式

参考文献的**类型**以单字母方式标识，具体如下：

**M**——专著  **C**——论文集  **N**——报纸文章  **J**——期刊文章  **D**——学位论文  **R**——报告

**Z**——其他

### 格式：

[**序号**]**作者.作品名称[类型]出处，时间：页码.**

**期刊：**[序号]作者.篇名[J].刊名，出版年份，卷号（期号）：起止页码.

**专著：**[序号]作者.书名[M].出版地：出版社，出版年份：起止页码.

**报告：**[序号]作者.篇名[R].出版地：出版者，出版年份：起始页码.

**论文集：**[序号]作者.篇名[C].出版地：出版者，出版年份：起始页码.

**学位论文：**[序号]作者.篇名[D].出版地：保存者，出版年份：起始页码.

**报纸：**[序号]作者.篇名[N].报纸名，出版日期（版次）.

**译著：**[序号]原著作者. 书名[M].译者，译.出版地：出版社，出版年份：起止页码.

**条例·：**[序号]颁布单位.条例名称[Z].发布日期

**英文参考文献格式与中文一致，但还要注意以下几点：**

- 作者姓名：**姓在前名在后，名字用单词首字母 姓与名之间用”，”隔开**

例如：Malcolm Richard Cowley 应为：Cowley, M.R.

- 如果有**多位作者**要用&隔开，后一位作者名在前姓在后
- 书名、报刊名使用斜体字

**示例：**

[3] Heider, E.R.& D.C.Oliver. The structure of color space in naming and memory of two languages [J]. Foreign Language Teaching and Research, 1999, (3): 62 – 67.
