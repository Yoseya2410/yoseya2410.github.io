# 结构

## 结构的语句格式

```c
struct date { 
int a;
char b;
double  c;
} var;
```

- `date`是结构名称，`var`是结构的变量名，`{}`中是结构的成员
- 函数内部声明的结构类型只能在函数内部使用
- 结构体可以用一个变量表达多个数据
- 声明了结构，就有了一种自己定义的数据类型`struct date`

## 结构的形式

**形式1：** 先声明结构类型，再定义结构变量
```c
//声明结构类型
struct date { 
int a;
int b;
};
struct date d1, d2;//定义结构变量
```

**形式2：** 同时定义结构类型和结构变量

- 结构可以不写名字
- d1和d2是这个结构形式的变量

```c
struct date { 
int a;
int b;
} d1, d2;
```

## 结构的初始化

**方法1：** 按照顺序初始化

```c
struct date var = {1,2,3};
```

**方法2：** 指定数据项初始化

```c
struct date var = {.a=1, .b=2, .c=3};
```

**方法3：** 声明结构类型并初始化

```c
struct date{
int a;
int b;
int c;
} var = {1,2,3};
```

## 结构成员
- 结构中的每一项都可以是不同类型的数据
- 在结构中使用`.`访问结构中的成员，例如`date.a`,表示访问结构`date`中的`a`

## 结构运算

- 要访问整个结构时，可以直接用结构变量的名字

```c
struct date { 
int a;
int b;
int c;
}var;

var =(struct date){1,3,5};
printf("%d %d %d", var.a, var.b, var.c);
```

通过结构变量给结构中的项初始化
- 结构允许赋值，取地址，向函数传递参数

**结构变量赋值**

```c
var.a = 15;
```

结构变量`var`给结构变量`vari`赋值

```c
//声明一个结构类型struct date
struct date { 
int a;
int b;
int c;
} var;

var =(struct date){1,3,5};//结构初始化
struct date vari;//定义变量，类型为struct date
vari = var;//把变量var赋值给vari
printf("%d %d %d", vari.a, vari.b, vari.c);
```

## 指向结构的指针

**指向结构变量**

```c
struct date *p = &var;
```

- `var`是结构变量名，`struct date`是指针类型
- 获取结构变量地址时要使用运算符`&`

**指向结构变量中的成员**

```c
(*p).a = 12;
```

或者用`->`表示指针所指的结构变量中的成员

```c
p->a =12;
```

## 结构作为函数参数

```c
int fun (struct date a);
```

- 整个结构可以作为参数的值传入函数
- 函数可以返回一个结构
- 如果要把结构传入函数，通常用传递指针的方式

示例：

```c
struct date { 
int a;
int b;
int c;
} var = {1,3,5};

void fun( struct date var ){
printf("%d\n", var.a);
}
fun(var);
```

## 结构数组

```c
struct date{
int a;
int b;
int c;
}class[3];
```
- `struct date`是结构体类型名,`class[3]`是结构体数组

**使用示例：**

- 结构数组的元素间用`,`隔开
- 结构数组元素`{}`中的三个值分别对应结构声明中的成员`a,b,c`

```c
struct date { 
int a;
int b;
int c;
};

struct date class[10]={
  {1,2,3}, //class[0]的值
  {4,5,6}  //class[1]的值
};
printf("%d",class[1].b);
```

输出结果：

```
5
```

- 结构体数组在定义的同时也可以初始化
- 当数组中全部元素都赋值时，可以不写数组长度

```c
struct date { 
int a;
int b;
int c;
} class[]={
  {1,2,3}, 
  {4,5,6}
};
```

**给结构数组中的元素赋值**

```c
class[1].a = 9;
```

**获取结构数组元素的值**

```c
int x = class[1].a;
```

 ## 自定义数据类型

 - 在C语言中可以使用`typedef`来定义一个新的数据类型名称
  
```c
typedef int abc;
```

这样就使abc成为了int类型的别名，现在可以通过这个新名字去定义变量

```c{1-2}
typedef int abc;
abc x = 1;

printf("%d\n", x);
printf("%d\n", sizeof(x));
```

输出结果：

```
1
4
```
这个自定义的`abc`类型和`int`类型一样也是4个字节，它们在性质上是一样的，不过是换了一个名字去使用`int`

`typedef`也可以用于结构，通过`typedef`可以简化结构类型的名称

```c
typedef struct date { 
int a;
int b;
int c;
} name;
name x; //定义一个name类型的变量，相当于struct date x
x.a=1; //给结构中的成员a赋值
printf("%d\n", x.a);
```

**结构数组使用typedef**

```c
typedef struct date { 
int a;
int b;
int c;
} class;

class x[10]={
  {1,2,3}, 
  {4,5,6} 
};
printf("%d",x[1].b);
```

## 联合

```c
union date
{
  int a;
  int b;
  int c;
} date;
```

- `union`中的每个成员共同使用同一块内存，但使用时只能储存其中一个成员的数据，各个成员的数据不能共存
- 所占内存大小等于最大的成员大小,内存大小为4(或者8)的整数倍，如果最大的成员大小不为4的整数倍时，内存大小会自动增大为4的整数倍(32位占4的整数倍，64位占8的整数倍)

```c
union date
{
  int i;
  float f;
  char str[20];
} date;
printf("%d", sizeof(date));
```
