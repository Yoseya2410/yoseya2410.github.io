# 条件表达式

这是C语言早期遗留下来的一种方式

运算符 说明
？ 如果为真则…
： 否则…

count>20 ? count - 10 : count + 10;

这个式子在C语言中表示，如果满足条件count>20则执行count-10否则执行count+10

相当于：

if (count > 20){
count - 10;
}else{
count + 10;
}
